# IGDownloader - Instagram Video Downloader

IGDownloader is a high-performance, single-page web application for downloading Instagram videos, reels, and photos. It features a design inspired by the popular **Snaptik** TikTok downloader but is fully styled with the vibrant **Instagram color palette**.

## Key Features
- **Snaptik Layout**: Familiar, simple, and effective user interface.
- **Instagram Aesthetics**: Modern gradients and a clean, premium look.
- **Lightning Fast**: Optimized for speed and responsiveness.
- **No Watermark**: Instructions and meta-data tailored for downloading IG content.
- **Fully Responsive**: Works perfectly on mobile, tablet, and desktop.

## Design System
- **Colors**: Pink-to-Purple gradients (`#f09433` to `#bc1888`)
- **Typography**: Apple System Fonts / Segoe UI for a native feel.
- **Icons**: FontAwesome 6 symbols.

## Getting Started
1. Open `index.html` in any web browser.
2. Paste an Instagram link into the input box.
3. Click "Download" to fetch the content.

##  Built With
- **HTML5**: Semantic structure.
- **CSS3**: Modern Flexbox, Grid, and Custom Properties.
- **JavaScript**: Pure, lightweight logic.
- **FontAwesome**: High-quality icons.
